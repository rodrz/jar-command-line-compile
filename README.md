# Create Jar with Prompt Console
Criar uma aplicação sem o uso de uma IDE.
Criar executáveis em windows console, permite manipular as dependências de forma controlada,
tornando o executáve com melhor portabilidade, impedindo que o IDE, 
coloque libraries atreladas ao server e ao IDE, se assemelhando melhor a um ?.exe
	1 - Compilação do Projeto
	2 - Explicação do arquivo manifesto cfvm
	3 - Adição de arquivos e pastas recursivamente
	4 - Compilação de um arquivo executável
	5 - Execução da aplicação ?.jar
